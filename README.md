# Service Meshes on AWS

Terraform module for deploying service meshes on [AWS](https://aws.amazon.com). This module uses
[Ansible](https://www.ansible.com/) for configuring all created resources.

## Requirements

On local machine:
* Python 3
* Ansible
* Terraform >= 0.12
